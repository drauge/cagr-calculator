# Investment Return Calculator

Static web calculator for GitLab Pages or any static host.

## What it supports

- Starting portfolio value
- Projection years
- Gross annualized return
- Annual fictitious-return tax assumption
- Recurring yearly contributions
- Recurring monthly contributions
- Optional lump-sum contributions by year/month
- Optional skipped contribution periods
- End-of-year net value summary
- CSV export

## Formula logic

The calculator uses a monthly simulation.

For every month:

```text
monthlyGrossReturn = (1 + grossAnnualReturn)^(1/12) - 1

balanceBeforeGrowth = startBalance + contributions
grossGrowth = balanceBeforeGrowth × monthlyGrossReturn
beforeTax = balanceBeforeGrowth + grossGrowth
```

In December only:

```text
tax = beforeTax × taxRate × fictitiousReturnBase
endBalance = beforeTax - tax
```

In other months:

```text
tax = 0
endBalance = beforeTax
```

Contributions are added at the start of the month, before monthly growth.

## Default scenario

The default values mirror the earlier example:

- €10,000 yearly from year 1 to year 3
- €16,500 yearly from year 4 to year 20
- yearly contribution month: January
- gross annual return: 10%
- tax: 36% × 5.88% annually

## GitLab Pages

Commit these files to your GitLab repository:

```text
index.html
styles.css
app.js
.gitlab-ci.yml
README.md
```

GitLab Pages will publish the static site from the `public/` folder generated by CI.
